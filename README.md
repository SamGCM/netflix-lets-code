## ⚙️ Executando o projeto 

Execute os comandos no seu terminal:

```bash
# Clone o repositório
$ git clone https://github.com/SamGCM/netflix-lets-code.git

# Entre no diretório
$ cd netflix-lets-code

# Instale as dependências
$ npm install

# Execute a aplicação
$ ng serve
```

Acesse em seu navegador http://localhost:4200/
